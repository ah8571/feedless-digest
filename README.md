# Feedless Digest

Feedless Digest is a newsletter project focused on finding thoughtful, high-signal writing from forums, blogs, and social platforms without forcing readers to spend hours inside algorithmic feeds.

The goal is simple:

- surface worthwhile long-form posts and deep discussions
- package them into clean weekly editions
- keep a public archive so past issues stay easy to revisit

This repository contains the public website, archive experience, and planning documents for the first version of the project.
