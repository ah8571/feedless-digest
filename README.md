# Feedless Digest

Feedless Digest is an attempt to build a better way to read the internet.

The problem is straightforward: a huge amount of the best writing now lives inside social feeds, forums, and community platforms, but finding it usually requires wading through addictive interfaces, noise, and endless scrolling.

This project is trying to solve that by turning high-signal long-form posts, threads, and discussions into a clean newsletter and public archive.

## What this project aims to do

- find worthwhile long-form writing across blogs, forums, and social platforms
- curate that material into readable newsletter editions
- maintain a public archive so strong posts remain easy to revisit
- separate discovery from the feed itself

## Why it may be interesting to open-source contributors

This is not just a marketing site. It is the beginning of a system for:

- discovering high-quality writing across fragmented platforms
- filtering signal from noise without recreating the addictive feed experience
- building a lightweight editorial pipeline for internet curation
- experimenting with archive-first publishing and newsletter workflows

If you care about information retrieval, media tooling, curation systems, newsletters, or healthier ways to consume online writing, this project should be relevant.

## Where contributions could be useful

- source discovery and ingestion workflows
- heuristics for ranking worthwhile posts and discussions
- archive and issue presentation
- signup, confirmation, and subscriber workflows
- self-hosted newsletter infrastructure
- editorial tools for reviewing and assembling issues

## Current state

Right now the repository contains:

- the public website
- the early archive experience
- signup flow groundwork
- infrastructure and setup notes for newsletter tooling
- project planning and source research docs
