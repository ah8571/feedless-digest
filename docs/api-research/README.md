# API Research Notes

Create one file per source here.

Suggested files:

- `reddit.md`
- `hacker-news.md`
- `x.md`
- `linkedin.md`
- `threads.md`
- `quora.md`

## Template

Use the following structure for each source:

### Source Summary
- What kind of long-form material can be found here?
- Why is it useful for Feedfree Digest?

### Access Path
- Official API, RSS, search index, Google Alerts, or other discovery path

### Full-Text Retrieval
- Can full content be retrieved reliably?
- If not, what is the fallback?

### Metadata Available
- Likes, comments, views, score, recency, tags, author, domain, or thread depth

### Constraints
- Rate limits
- Pricing
- Auth requirements
- Terms-of-use risk

### MVP Heuristics
- What simple rules define a likely worthwhile item?

### Recommendation
- Build now, defer, or use only as a discovery source
